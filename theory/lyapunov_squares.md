# Анализ устойчивости модели Шаповалова с помощью функции Ляпунова

## Система

$$
\begin{cases}
\dot{x} = -\sigma x + \delta y \\
\dot{y} = \mu(x + y) - \beta x z \\
\dot{z} = -\gamma z + \alpha x y
\end{cases}
$$

## Функция Ляпунова

$$
H(x,y,z) = \frac{1}{2}(A x^2 + B y^2 + C z^2), \quad A,B,C>0.
$$

Градиент: \(\nabla H = (A x, B y, C z)\).

## Производная \(\dot{H}\)

$$
\dot{H} = \langle \nabla H, f \rangle = -\sigma A x^2 + \mu B y^2 - \gamma C z^2 + (\delta A + \mu B)xy + k\, xyz,
$$

где \(k = \alpha C - \beta B\).

## Разбор случаев по теоремам Ляпунова

### 1. Случай \(k = 0\) (\(\alpha C = \beta B\))

$$
\dot{H} = -\sigma A x^2 + \mu B y^2 - \gamma C z^2 + (\delta A + \mu B)xy.
$$

Квадратичная форма по переменным \(x,y\):

$$
\begin{pmatrix}
-\sigma A & \dfrac{\delta A + \mu B}{2} \\
\dfrac{\delta A + \mu B}{2} & \mu B
\end{pmatrix}.
$$

**Анализ главных миноров (критерий Сильвестра):**

- Первый главный минор: \(\Delta_1 = -\sigma A < 0\) (выполнено).
- Второй главный минор: \(\Delta_2 = (-\sigma A)(\mu B) - \left(\dfrac{\delta A + \mu B}{2}\right)^2 < 0\) **не выполняется** (из-за положительного члена \(\mu B > 0\)).

По **прямой теореме Ляпунова** (требуется \(\dot{H} \le 0\)) условие **не выполняется**.  
Нельзя доказать ни устойчивость, ни асимптотическую устойчивость равновесий.

### 2. Случай \(k > 0\) (\(\alpha C > \beta B\))

В любой окрестности точки \(O_0(0,0,0)\) вдоль направления \(x=0\), \(z=0\), \(y \neq 0\):

$$
\dot{H}(0,y,0) = \mu B y^2 > 0.
$$

Условие \(\dot{H} \le 0\) **не выполняется**.  
По **прямой теореме Ляпунова** нельзя доказать ни устойчивость, ни асимптотическую устойчивость.

**По теореме Четаева**: в области \(V = \{(x,y,z) \mid y > 0\}\) (малая окрестность \(O_0\)) имеем \(H > 0\) и \(\dot{H} > 0\) вблизи \(O_0\). Следовательно, точка \(O_0\) **неустойчива**.

### 3. Случай \(k < 0\) (\(\alpha C < \beta B\))

В любой окрестности точки \(O_0(0,0,0)\) вдоль направления \(x=0\), \(z=0\), \(y \neq 0\):

$$
\dot{H}(0,y,0) = \mu B y^2 > 0.
$$

Условие \(\dot{H} \le 0\) **не выполняется**.  
По **прямой теореме Ляпунова** нельзя доказать ни устойчивость, ни асимптотическую устойчивость.  
По **теореме Четаева** также не удаётся строго доказать неустойчивость во всей окрестности.

## Итог

Во всех трёх случаях функция \(H\) **не позволяет** применить прямую теорему Ляпунова для доказательства (асимптотической) устойчивости равновесий.  
Только в случае \(k > 0\) теорема Четаева позволяет доказать **неустойчивость** точки \(O_0(0,0,0)\).
